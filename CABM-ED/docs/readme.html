项目理念

CABM-ED 是一款 AI 驱动的养成类游戏，核心理念与 CABM（Code Afflatus & Beyond Matter） 一致。

「当灵性注入载体，它便挣脱物质躯壳，抵达超验之境」

故事始于你将她带回家的那一刻。面对这个无家可归、沉默寡言的她，你唯一能做的，就是提供一个可以称之为「家」的地方。而你很快发现，这个家也因为她的存在，开始充满了意想不到的生机……

在这个项目中，你将遇见一位拥有自主意识的「她」——即使你不在线，她也会在自己的世界里继续生活、成长、经历。没有重置，没有清除，每一个选择都将成为她生命轨迹中不可磨灭的印记。

作者不鼓励通过修改存档、修改配置文件、提示词注入等方式干涉「她」。虽然也管不了……

角色故事

雪狐「SnowFox」
  • CABM-ED背景故事（有刀子）
  • 小说（与游戏剧情无关）

平台支持

  • ✅ 完整支持（功能完整、持续更新）：Windows、Android
  • ⚠️ 基础支持（更新较慢、缺乏维护）：Linux、macOS、Web
  • ❌ 不支持（可使用 Web 版替代）：iOS、HarmonyOS

功能玩法

🗣️ 对话系统

  • 接入大语言模型（LLM），支持自然流畅的对话
  • 情感系统：角色拥有多种情感状态（开心、难过、生气、平静等），由语言模型控制
  • 好感度与交互意愿：好感度范围 -65536～65535，交互意愿范围 0～100，由语言模型控制
  • 语音对话：支持中、英、日三种语言
  • 主动对话：长时间无操作时，雪狐可能会主动与你交流

🧠 记忆系统

  • 突破上下文窗口限制，实现长期记忆存储
  • 详细说明请参阅：关于记忆力的说明

🏠 多场景互动

  • 点击屏幕右侧可切换不同场景
  • 雪狐在每个场景中都有不同位置和姿势

📔 日记系统

  • 所有对话都会以雪狐的身份被记录
  • 支持查看完整对话记录和语音回放
  • 离线期间雪狐也会自动写日记
  • 其他生活琐事也会被记录

👗 换装系统

  • 入口：浴室镜子
  • 提供多套服装，可自由切换
  • 每套服装在不同场景和对话中都有独特表现

🗺️ 探索模式（Preview）

  • 入口：玄关/天台/超市打开地图
  • 功能特色：
    • 物资箱：随机刷新物品
    • 雪狐：可拾取物品，主动攻击敌人，你也可以和她聊天
    • 敌人：主动攻击，击败后掉落物品
  • 地图每日刷新

📚 故事模式（Preview）

  • 入口：书房电脑
  • 和你开启一段「如我所书」的故事
  • 可以从任意节点创造分支

🌀 阈界回廊（大饼）

  • 入口：地图

🌱 休闲玩法

  • 游戏（入口：客厅点击雪狐）：和雪狐下五子棋、中国象棋
  • 菜园（入口：天台地面）：种植和收获作物，享受养老式玩法
  • 烹饪（入口：厨房灶台）：自由组合食材进行烹饪
  • 超市（入口：超市门）：物品交换，每日刷新
